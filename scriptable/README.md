# Scriptable 小组件使用说明

1. 在 iPhone 安装 **Scriptable**。
2. 从网页点击“导出配置”，把 JSON 文件保存到 iPhone 的“文件”App。
3. 在 Scriptable 新建脚本，分别粘贴导入脚本和组件脚本内容，并以对应名称保存。
4. 在 Scriptable 运行“净值望远镜-导入配置”，从“文件”App 选择第 2 步的 JSON 文件。脚本只保留代码与名称，不导入持有金额。
5. 长按 iPhone 桌面，添加 Scriptable 小组件，选择“净值望远镜-210日回撤组件”。
6. 在小组件参数中填写：
   - `holding`：显示导入顺序靠前的持仓基金；小号显示 1 只，中号显示 3 只。
   - `watch`：显示观察基金。
   - `holding:100039`：只显示指定持仓基金。
   - `watch:512400,513770,017731`：中号显示指定的 3 只观察基金。

小组件最多每 6 小时主动请求一次净值；iOS 可能延后刷新。更新网页基金列表后，重新导出并运行一次导入脚本即可同步。
