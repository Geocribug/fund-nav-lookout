# Scriptable 小组件使用说明

1. 在 iPhone 安装 **Scriptable**。
2. 从网页点击“导出配置”，把 JSON 文件保存到 iPhone 的“文件”App。
3. 在 Scriptable 新建脚本，分别粘贴 `净值望远镜-导入配置.js` 和 `净值望远镜-210日回撤组件.js` 的内容，并以对应名称保存。
4. 在 Scriptable 运行“净值望远镜-导入配置”，从“文件”App 选择第 2 步的 JSON 文件。脚本只保留代码与名称，不导入持有金额。
5. 长按 iPhone 桌面，添加两个**大型** Scriptable 小组件，均选择“净值望远镜-210日回撤组件”。
6. 分别在两个小组件参数中填写：
   - `holding`：显示导入顺序靠前的 7 只持仓基金。
   - `watch`：显示导入顺序靠前的 7 只观察基金。
   - `holding:100039,005827,161725`：只显示指定持仓基金，代码顺序就是展示顺序。
   - `watch:512400,513770,017731`：只显示指定观察基金，代码顺序就是展示顺序。

小号显示 1 只，中号显示 3 只，大型显示 7 只。若要为两个组件分别手动排序，请在各自的参数中按希望的顺序填写基金代码。

在 iPhone 长按组件并选择“编辑小组件”，直接在 **Parameter** 输入框填写上述内容即可；英文逗号或中文逗号都可以。

小组件最多每 6 小时主动请求一次净值；iOS 可能延后刷新。更新网页基金列表后，重新导出并运行一次导入脚本即可同步。

## 策略组合再平衡组件

1. 新建并粘贴 `策略再平衡-配置.js`，先运行它；选择网页导出的配置 JSON，随后逐个填写每个策略组合的当前金额（元，支持两位小数）。金额只保存在 iPhone 本机。
2. 新建并粘贴 `策略再平衡-组件.js`，在桌面添加一个**中型** Scriptable 小组件并选择它。
3. 默认展示前 3 个策略组合；在 Parameter 中填写策略名称可筛选并排序，例如：`美股科技,全球主动,核心固收`。

组件会显示策略名称、当前金额、当前 / 目标占比、偏离值、仅新增资金所需金额，以及不增资时的买入 / 卖出清单。仅新增资金和不增资调仓均按全部策略的目标合计归一化计算；结果不含申购赎回费、到账时间和限购约束。
